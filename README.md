# 📁 Core Programming & Computer Science Exercises

A collection of programming tasks focused on algorithms, Object-Oriented Programming (OOP) concepts, data structures, and file I/O operations.

## 🛠️ Repository Structure

* **`01_bibliophiles`** — **Data Structures & Sets**: A recommendation system analyzing book lovers' preferences using set operations (unions, intersections, differences).
* **`02_geometry`** — **Console Parser & Math**: A dynamic console application that parses user inputs, classifies geometric shapes, and calculates areas/volumes in a `while` loop.
* **`03_modulo`** — **Algorithms & Dictionaries**: A script that groups collections of numbers into custom dictionaries based on mathematical modulo (`%`) operations.
* **`04_distance`** — **Advanced OOP & Operator Overloading**: A custom `Dist` class handling metrics (mm, cm, m, km) with fully overloaded arithmetic operators (`+`, `-`, `*`, `/`).
* **`05_iterator`** — **Design Patterns & Custom Iterators**: Implementation of a custom iterable class (`IterClass`) that filters collections using predicates and lambda functions.
* **`06_hobbit`** — **File I/O & Text Analytics**: A comprehensive text analyzer that processes *The Hobbit* novel, calculating word frequencies, chapter metrics, and filtering stopwords.

## 💻 Tech Stack
* **Language**: Python (or Java/C++ depending on what you used)
* **Concepts**: OOP, Design Patterns (Iterator, MVC), File I/O, Data Analytics, Algorithms.
